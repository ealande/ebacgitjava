Podemos considerar o método de programação procedural como sendo um dos primeiros métodos que foram desenvolvidos para a criação de softwares, a partir desse método o desenvolvedor aglutina os métodos, funções, variáveis, etc., imprescindíveis para o funcionamento do software. Alguns desenvolvedores mencionam que esse método pode gerar códigos de difícil compreensão, dificultando a manutenção e escabilidade do código.
Já o método de programação orientada a objetos é uma abordagem mais moderna, onde diferentes arquivos con:tendo as classes e objetos são criados, cada um contendo o código “setorizado” de cada objeto e classe. Essa abstração das classes permite que o código seja compreendido mais facilmente, pois reflete a divisão dos objetos que existem na “vida real”, viabilizando, assim, a manutenção do código.
